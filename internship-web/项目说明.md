# 高校实习管理系统 - 前端项目说明文档

## 📋 项目简介

高校实习过程跟踪与评价管理系统的前端项目，采用现代化的 Vue 3 技术栈开发，提供完整的实习管理、评价管理、校企协同等功能模块。

## 🛠️ 技术栈

### 核心框架
- **Vue 3.5.13** - 渐进式 JavaScript 框架，使用 Composition API
- **Vite 6.0.0** - 下一代前端构建工具，提供极速的开发体验

### UI 组件库
- **Element Plus 2.8.8** - 基于 Vue 3 的组件库
- **@element-plus/icons-vue 2.3.1** - Element Plus 图标库

### 状态管理与路由
- **Pinia 2.1.7** - Vue 官方推荐的状态管理库
- **Vue Router 4.2.5** - Vue.js 官方路由管理器

### HTTP 请求
- **Axios 1.6.2** - 基于 Promise 的 HTTP 客户端

### 数据可视化
- **ECharts 5.6.0** - 强大的数据可视化图表库
- **vue-echarts 6.7.3** - Vue 3 的 ECharts 封装组件

### 富文本编辑
- **Quill 1.3.7** - 现代化的富文本编辑器

### 开发工具
- **@vitejs/plugin-vue 5.2.1** - Vite 的 Vue 插件
- **ESLint 8.56.0** - JavaScript 代码检查工具
- **eslint-plugin-vue 9.19.2** - Vue 的 ESLint 插件
- **Prettier 3.1.1** - 代码格式化工具
- **@vue/eslint-config-prettier 9.0.0** - Vue 的 Prettier 配置

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 生产环境构建
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

### 环境变量配置
创建 `.env` 文件（如不存在）：
```env
VITE_PORT=5173
VITE_API_BASE_URL=http://localhost:8080
```

### 1. 端口被占用
修改 `.env` 文件中的 `VITE_PORT` 配置

### 2. API 请求失败
检查 `.env` 文件中的 `VITE_API_BASE_URL` 配置是否正确
确保后端服务已启动

### 3. 依赖安装失败
清除缓存后重新安装：
```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```
