# 图表集成步骤说明

## 第一步：安装依赖

在项目根目录 `internship-web/` 下运行：

```bash
npm install echarts vue-echarts
```

或者使用 yarn：

```bash
yarn add echarts vue-echarts
```

**注意**：安装完成后需要重启开发服务器。

## 第二步：验证安装

安装成功后，`package.json` 中应该包含以下依赖：

```json
{
  "dependencies": {
    "echarts": "^5.4.3",
    "vue-echarts": "^6.6.9"
  }
}
```

## 第三步：测试运行

1. 启动后端服务（确保统计接口可用）
2. 启动前端开发服务器：`npm run dev`
3. 登录系统并访问仪表盘页面
4. 验证图表是否正常显示

## 图表组件使用说明

所有图表组件都已配置好：
- **PieChart** - 饼图（用于实习进度、岗位类型分布）
- **BarChart** - 柱状图（用于人数对比、分数分布）
- **LineChart** - 折线图（用于趋势分析）
- **RadarChart** - 雷达图（用于个人多维度评价）

图表会自动：
- 适配容器大小
- 显示空数据提示
- 使用柔和的配色方案

## 故障排查

### 如果图表不显示

1. 检查是否安装了依赖
2. 检查浏览器控制台是否有错误
3. 检查后端API是否正常返回数据
4. 检查数据格式是否正确

### 如果数据格式错误

检查后端返回的数据格式是否匹配前端图表组件期望的格式：
- 饼图：`{ name, value, color }`
- 柱状图：`{ name, value, color }`
- 折线图：`{ name, value }` 或 `{ month, averageDays }`
- 雷达图：需要 `indicators` 和 `data` 两个数组
