# 实习管理系统 - 后端项目说明

## 项目概述

实习管理系统后端服务，基于 Spring Boot 框架开发，提供实习管理相关的 RESTful API 接口。

## 技术栈

### 核心框架
- **Spring Boot**: 2.7.6
- **Java**: 1.8
- **Maven**: 项目构建工具

### Web 框架
- **Spring Boot Web**: Web 应用开发
- **Spring Boot Security**: 安全认证与授权
- **Spring MVC**: MVC 框架

### 数据持久化
- **MyBatis Plus**: 3.5.3 - ORM 框架
- **MySQL**: 8.0.33 - 关系型数据库
- **MySQL Connector**: 数据库驱动

### 缓存
- **Spring Boot Data Redis**: Redis 缓存支持

### 认证与授权
- **JWT (JSON Web Token)**: 0.9.1 - 无状态身份认证

### API 文档
- **Swagger (Springfox)**: 3.0.0 - API 文档生成工具

### 工具库
- **Lombok**: 简化 Java 代码
- **Jackson**: JSON 数据处理
- **Apache POI**: 5.2.3 - Excel 文件处理
- **Spring Boot Validation**: 数据验证

### 测试
- **Spring Boot Test**: 单元测试
- **Spring Security Test**: 安全测试
