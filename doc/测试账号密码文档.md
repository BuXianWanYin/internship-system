## 九、账号使用说明

### 9.1 登录方式

1. 访问系统登录页面
2. 输入用户名和密码（默认密码：`123456`）
3. 点击登录按钮
4. 登录成功后，系统会根据用户角色显示相应的功能菜单

### 9.2 密码修改

1. 登录系统后，进入个人中心
2. 选择"修改密码"功能
3. 输入原密码和新密码
4. 确认修改

### 9.3 账号状态

- **状态 = 1**：账号已启用，可以正常登录
- **状态 = 0**：账号已禁用，无法登录

所有测试账号的状态均为已启用（status = 1）。

## 十、注意事项

1. **密码安全**
   - 所有测试账号使用统一的默认密码：`123456`
   - 生产环境部署前，请务必修改所有账号密码
   - 建议使用强密码策略

2. **数据权限**
   - 不同角色的用户只能查看和管理权限范围内的数据
   - 学校管理员只能管理本校数据
   - 班主任只能管理本班学生数据
   - 企业管理员只能管理本企业数据

3. **测试数据**
   - 测试账号包含完整的测试数据
   - 测试过程中可以正常使用所有功能
   - 建议在测试环境使用，避免影响生产数据

4. **账号管理**
   - 系统管理员可以管理所有账号
   - 学校管理员可以管理本校账号
   - 建议定期检查账号状态和权限
