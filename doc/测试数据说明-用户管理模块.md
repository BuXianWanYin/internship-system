## 八、数据验证SQL

### 8.1 验证用户总数
```sql
SELECT COUNT(*) as total_users FROM user_info WHERE delete_flag = 0;
```

### 8.2 验证角色分配
```sql
SELECT r.role_name, COUNT(ur.user_id) as user_count
FROM role_info r
LEFT JOIN user_role ur ON r.role_id = ur.role_id AND ur.delete_flag = 0
WHERE r.delete_flag = 0
GROUP BY r.role_id, r.role_name
ORDER BY r.role_id;
```

### 8.3 验证班级班主任关联
```sql
SELECT c.class_name, u.real_name as teacher_name
FROM class_info c
LEFT JOIN user_info u ON c.class_teacher_id = u.user_id
WHERE c.delete_flag = 0;
```

### 8.4 验证学生班级关联
```sql
SELECT c.class_name, COUNT(s.student_id) as student_count
FROM class_info c
LEFT JOIN student_info s ON c.class_id = s.class_id AND s.delete_flag = 0
WHERE c.delete_flag = 0
GROUP BY c.class_id, c.class_name;
```
