### 3. 下一步操作

#### 3.1 后端
1. 安装依赖：
   ```bash
   cd internship-server
   mvn clean install
   ```

2. 创建数据库：
   - 创建MySQL数据库：`internship_db`
   - 字符集：`utf8mb4`
   - 排序规则：`utf8mb4_unicode_ci`

3. 修改数据库配置：
   - 编辑 `src/main/resources/application.yml`
   - 修改数据库连接信息（url、username、password）

4. 启动Redis服务（如果未启动）

#### 3.2 前端
1. 安装依赖：
   ```bash
   cd internship-web
   npm install
   ```

2. 创建环境变量文件：
   ```bash
   cp .env.example .env
   ```
   然后根据实际情况修改 `.env` 文件中的配置

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 注意事项

1. **数据库配置**：确保MySQL 8已安装并运行，数据库连接信息正确
2. **Redis配置**：确保Redis已安装并运行（默认端口6379）
3. **端口占用**：确保8080（后端）和3000（前端）端口未被占用
4. **JWT密钥**：生产环境必须修改 `application.yml` 中的JWT密钥
5. **Swagger访问**：开发环境可通过 `http://localhost:8080/api/swagger-ui.html` 访问API文档

## 验证环境

### 后端验证
1. 启动Spring Boot应用
2. 访问 `http://localhost:8080/api/swagger-ui.html` 查看Swagger文档
3. 检查日志，确认无错误

### 前端验证
