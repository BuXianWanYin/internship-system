# 实习管理系统模块操作说明总览

## 文档说明

本文档是实习管理系统各模块操作说明的总览文档，详细的操作说明请参考对应的模块文档。

## 模块列表

### 1. [实习申请模块操作说明](./01-实习申请模块操作说明.md)
- **功能**：管理学生的实习申请，包括合作企业实习和自主实习两种类型
- **主要流程**：
  - 合作企业实习：学生提交申请 → 学校审核 → 企业筛选 → 面试 → 录用确认 → 学生确认上岗 → 实习中 → 实习结束 → 评价完成
  - 自主实习：学生提交申请 → 班主任审核 → 自动确认上岗 → 实习中 → 实习结束
- **关键角色**：学生、学校管理员、学院领导、班主任、企业管理员、企业导师

### 2. [实习计划模块操作说明](./02-实习计划模块操作说明.md)
- **功能**：管理学校的实习计划，包括计划的创建、审核、发布等
- **主要流程**：创建草稿 → 提交审核 → 审核计划 → 发布计划
- **关键角色**：系统管理员、学校管理员、学院领导、班主任

### 3. [实习岗位模块操作说明](./03-实习岗位模块操作说明.md)
- **功能**：管理企业的实习岗位信息，包括岗位的创建、审核、发布等
- **主要流程**：创建岗位（待审核） → 审核岗位（通过后自动发布） → 关闭岗位
- **关键角色**：企业管理员、系统管理员、学校管理员

### 4. [面试模块操作说明](./04-面试模块操作说明.md)
- **功能**：管理学生的面试信息，包括面试的创建、确认、结果录入等
- **主要流程**：企业安排面试 → 学生确认面试 → 完成面试 → 录入面试结果
- **关键角色**：企业管理员、企业导师、学生

### 5. [考勤模块操作说明](./05-考勤模块操作说明.md)
- **功能**：管理学生的实习考勤记录，包括考勤组的创建、考勤规则的设置、考勤记录的录入和审批等
- **主要流程**：创建考勤组 → 设置考勤规则 → 添加学生 → 录入考勤记录 → 审批考勤记录
- **关键角色**：企业管理员、企业导师、学生、班主任

### 6. [实习日志周报成果反馈模块操作说明](./06-实习日志周报成果反馈模块操作说明.md)
- **功能**：管理学生的实习日志、周报、成果和反馈
- **主要流程**：
  - 日志/周报/成果：学生提交 → 审批 → 通过/拒绝
  - 反馈：学生提交 → 企业/学校回复
- **关键角色**：学生、企业管理员、企业导师、班主任、学校管理员

### 7. [评价模块操作说明](./07-评价模块操作说明.md)
- **功能**：管理实习评价，包括自我评价、企业评价、学校评价和综合评分
- **主要流程**：提交自我评价 → 提交企业评价 → 提交学校评价 → 自动计算综合评分
- **关键角色**：学生、企业管理员、企业导师、班主任、学校管理员

### 8. [合作管理模块操作说明](./08-合作管理模块操作说明.md)
- **功能**：管理学校与企业的合作关系，包括校企合作申请的提交、审核、合作关系的建立和维护等
- **主要流程**：企业提交合作申请 → 学校审核 → 建立合作关系 → 维护合作关系 → 终止合作关系
- **关键角色**：企业管理员、系统管理员、学校管理员

### 9. [用户管理模块操作说明](./09-用户管理模块操作说明.md)
- **功能**：管理系统中的各类用户，包括学生、教师、企业、企业导师、学校管理员等
- **主要功能**：
  - 学生管理：批量导入、手动添加、更新、删除
  - 教师管理：添加、更新、删除
  - 企业管理：企业注册（无需审核）、添加、更新、删除
  - 企业导师管理：添加、更新、删除
  - 学校管理员管理：添加、更新、删除
- **关键角色**：系统管理员、学校管理员、企业管理员

### 10. [系统管理模块操作说明](./10-系统管理模块操作说明.md)
- **功能**：管理系统的基础数据和配置，包括学校、学院、专业、班级、学期等基础信息的管理，以及系统配置的管理
- **主要功能**：
  - 学校管理：添加、更新、删除
  - 学院管理：添加、更新、删除
  - 专业管理：添加、更新、删除
  - 班级管理：添加、更新、删除
  - 学期管理：添加、更新、设置当前学期、删除
  - 系统配置管理：查看、更新系统配置、评价权重配置、考勤规则配置
- **关键角色**：系统管理员、学校管理员

## 企业注册到发布岗位完整流程

### 流程图

```
企业注册
    ↓
[企业管理员] 填写企业信息 → 提交注册 → 注册成功 → 可以立即登录系统
    ↓
[企业管理员] 登录系统 → 提交校企合作申请
    ↓
[系统管理员/学校管理员] 审核合作申请
    ↓
审核通过 → 自动建立合作关系
    ↓
[企业管理员] 创建实习岗位（草稿）
    ↓
[企业管理员] 提交岗位审核
    ↓
[系统管理员/学校管理员] 审核岗位
    ↓
审核通过 → 岗位自动发布（状态变为"已发布"）
    ↓
学生可以看到并申请岗位
```

### 详细步骤说明

#### 第一步：企业注册
- **操作角色**：企业管理员
- **操作内容**：填写企业基本信息（企业名称、统一社会信用代码、地址、联系人等）
- **注册结果**：**注册成功后，企业管理员可以立即登录系统，无需等待审核**
- **详细说明**：见 [用户管理模块操作说明](./09-用户管理模块操作说明.md) 第4.1节

#### 第二步：提交校企合作申请
- **操作角色**：企业管理员（已注册并登录）
- **前置条件**：企业注册成功，企业管理员已登录系统
- **操作内容**：
  - 选择要合作的学校
  - 填写合作类型、合作时间、合作描述等信息
  - 提交合作申请
- **提交后状态**：待审核
- **详细说明**：见 [合作管理模块操作说明](./08-合作管理模块操作说明.md) 第1.2节

#### 第四步：审核校企合作申请
- **操作角色**：系统管理员、学校管理员
- **审核内容**：审核企业资质、合作类型、合作时间、合作描述等
- **审核结果**：
  - **通过**：自动建立合作关系，企业可以发布岗位
  - **拒绝**：需要填写拒绝原因，企业可以修改后重新提交
- **详细说明**：见 [合作管理模块操作说明](./08-合作管理模块操作说明.md) 第1.3节

#### 第五步：创建实习岗位
- **操作角色**：企业管理员（已建立合作关系）
- **前置条件**：
  - 企业已与学校建立合作关系
  - 存在已发布的实习计划（岗位需要关联实习计划）
- **操作内容**：
  - 填写岗位信息（岗位名称、岗位类型、岗位描述、招聘人数、工作地点、薪资待遇、实习时间等）
  - 关联已发布的实习计划
  - 保存为草稿
- **详细说明**：见 [实习岗位模块操作说明](./03-实习岗位模块操作说明.md) 第2.2节

#### 第六步：提交岗位审核
- **操作角色**：企业管理员
- **操作内容**：确认岗位信息无误后，提交审核
- **提交后状态**：待审核
- **详细说明**：见 [实习岗位模块操作说明](./03-实习岗位模块操作说明.md) 第2.2节

#### 第七步：审核岗位
- **操作角色**：系统管理员、学校管理员
- **审核内容**：审核岗位信息是否合理、是否符合要求
- **审核结果**：
  - **通过**：岗位状态自动变为"已发布"，学生可以查看并申请
  - **拒绝**：需要填写拒绝原因，企业可以修改后重新提交
- **详细说明**：见 [实习岗位模块操作说明](./03-实习岗位模块操作说明.md) 第2.2节

#### 第八步：学生查看岗位
- **岗位可见条件**（必须同时满足）：
  1. 岗位状态为"已发布"
  2. 企业已与学校建立合作关系
  3. 岗位关联的实习计划已发布
  4. 岗位的实习时间在有效期内
  5. 岗位招聘人数未满（如果设置了招聘人数）
- **详细说明**：见 [实习岗位模块操作说明](./03-实习岗位模块操作说明.md) 第6节

### 关键要点

1. **企业注册无需审核**：企业注册成功后可以立即登录系统，无需等待审核
2. **合作申请需要审核**：企业登录后需要提交校企合作申请，合作申请需要学校审核通过后才能建立合作关系
3. **合作关系是必要条件**：只有建立了合作关系，企业才能发布岗位，学生才能看到岗位
3. **岗位需要关联实习计划**：岗位必须关联一个已发布的实习计划
4. **岗位审核通过即发布**：岗位审核通过后自动变为"已发布"状态，无需手动发布
5. **学生可见性有多个条件**：学生能看到岗位需要满足多个条件，不仅仅是"已发布"

## 完整实习流程

### 合作企业实习完整流程

1. **准备阶段**
   - 系统管理员/学校管理员：创建学校、学院、专业、班级、学期等基础数据
   - 系统管理员/学校管理员：导入学生、添加教师
   - **企业管理员：注册企业账号，填写企业基本信息，提交注册**
     - **注册成功后，企业管理员可以立即登录系统，无需等待审核**
   - **企业管理员：登录系统后，提交校企合作申请**
     - 填写合作信息：合作类型、合作开始时间、合作结束时间、合作描述等
     - 提交合作申请，等待学校审核
   - **系统管理员/学校管理员：审核校企合作申请**
     - 审核通过：自动建立合作关系，企业可以发布实习岗位
     - 审核拒绝：企业可以修改后重新提交合作申请

2. **计划阶段**
   - 系统管理员/学校管理员/学院领导：创建实习计划，提交审核
   - 系统管理员/学校管理员：审核实习计划，发布计划
   - **企业管理员：创建实习岗位（草稿状态）**
     - 填写岗位信息：岗位名称、岗位类型、岗位描述、招聘人数、工作地点、薪资待遇、实习时间等
     - 关联已发布的实习计划
     - 保存为草稿
   - **企业管理员：提交岗位审核**
     - 确认岗位信息无误后，提交审核
     - 岗位状态变为"待审核"
   - **系统管理员/学校管理员：审核实习岗位**
     - 审核通过：岗位状态自动变为"已发布"，学生可以查看并申请
     - 审核拒绝：企业可以修改岗位信息后重新提交审核
   - **岗位发布后，学生可见条件：**
     - 岗位状态为"已发布"
     - 企业已与学校建立合作关系
     - 岗位关联的实习计划已发布
     - 岗位的实习时间在有效期内
     - 岗位招聘人数未满（如果设置了招聘人数）

3. **申请阶段**
   - 学生：浏览岗位，提交实习申请
   - 学校管理员/学院领导/班主任：审核学生申请
   - 企业管理员：筛选申请（标记感兴趣、安排面试、录用、拒绝）
   - 企业管理员：安排面试
   - 学生：确认面试
   - 企业管理员/企业导师：完成面试，录入面试结果
   - 企业管理员：录用学生
   - 学生：确认上岗
   - 企业管理员：为学生分配企业导师

4. **实习阶段**
   - 企业管理员：创建考勤组，设置考勤规则
   - 学生：提交考勤记录
   - 企业管理员/企业导师：审批考勤记录
   - 学生：提交实习日志、周报、成果
   - 企业管理员/企业导师：审批日志、周报、成果
   - 学生：提交反馈
   - 企业管理员/企业导师/班主任：回复反馈

5. **结束阶段**
   - 系统管理员/学校管理员/学院领导/班主任/企业管理员/企业导师：结束实习
   - 学生：提交自我评价
   - 企业管理员/企业导师：提交企业评价
   - 班主任/学校管理员：提交学校评价
   - 系统：自动计算综合评分

### 自主实习完整流程

1. **准备阶段**
   - 系统管理员/学校管理员：创建学校、学院、专业、班级、学期等基础数据
   - 系统管理员/学校管理员：导入学生、添加教师

2. **申请阶段**
   - 学生：提交自主实习申请
   - 班主任：审核学生申请（通过/拒绝）
   - 系统：自动确认上岗（审核通过后）

3. **实习阶段**
   - 学生：提交实习日志、周报、成果
   - 班主任：审批日志、周报、成果
   - 学生：提交反馈
   - 班主任：回复反馈

4. **结束阶段**
   - 系统管理员/学校管理员/学院领导/班主任：结束实习
   - 学生：提交自我评价
   - 班主任/学校管理员：提交学校评价
   - 系统：自动计算综合评分

## 角色权限总览

### 系统管理员
- 所有模块的完全权限
- 系统配置管理
- 用户管理
- 基础数据管理

### 学校管理员
- 实习申请审核
- 实习计划管理
- 实习岗位审核
- 用户管理（学生、教师）
- 基础数据管理（学院、专业、班级、学期）
- 合作管理
- 评价管理
- 统计查看

### 学院领导
- 本学院实习申请审核
- 本学院实习计划管理
- 本学院学生管理
- 本学院评价管理
- 本学院统计查看

### 班主任
- 本班级实习申请审核（包括自主实习）
- 本班级实习计划管理
- 本班级学生管理
- 本班级学生日志/周报/成果审批（自主实习）
- 本班级学生评价管理
- 本班级统计查看

### 企业管理员
- 企业信息管理
- 企业导师管理
- 实习岗位管理
- 实习申请筛选
- 面试管理
- 考勤管理
- 学生日志/周报/成果审批
- 企业评价管理
- 企业统计查看

### 企业导师
- 分配给自己的学生的日志/周报/成果审批
- 分配给自己的学生的考勤管理
- 分配给自己的学生的企业评价
- 分配给自己的学生的面试结果录入

### 学生
- 实习申请提交
- 面试确认
- 上岗确认
- 考勤打卡
- 日志/周报/成果提交
- 反馈提交
- 自我评价提交
- 个人信息管理
