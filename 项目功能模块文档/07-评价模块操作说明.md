# 评价模块操作说明

## 模块概述

评价模块用于管理实习评价，包括自我评价、企业评价、学校评价和综合评分。评价是实习管理的重要环节，用于评估学生的实习表现。

## 一、评价类型说明

### 1.1 自我评价
- **评价人**：学生
- **评价对象**：学生自己
- **评价内容**：学生对自身实习表现的自我评价

### 1.2 企业评价
- **评价人**：企业管理员、企业导师
- **评价对象**：学生
- **评价内容**：企业对学生在实习期间的表现评价

### 1.3 学校评价
- **评价人**：班主任、学校管理员
- **评价对象**：学生
- **评价内容**：学校对学生实习表现的评价

### 1.4 综合评分
- **计算方式**：系统根据自我评价、企业评价、学校评价自动计算
- **评分规则**：根据系统配置的权重计算

## 二、自我评价流程

### 2.1 提交自我评价

- **操作角色**：学生
- **前置条件**：
  - 实习已结束或即将结束
  - 学生已完成实习
- **操作说明**：
  1. 进入自我评价页面
  2. 点击"新建评价"
  3. 填写评价信息：
     - 实习总结
     - 工作表现自评
     - 学习收获
     - 能力提升
     - 不足之处
     - 改进方向
     - 评分（各项指标）
  4. 提交评价
- **状态**：待确认

### 2.2 修改自我评价

- **操作角色**：学生
- **前置条件**：
  - 评价状态为"待确认"或"已提交"
  - 在允许修改的时间范围内
- **操作说明**：
  1. 找到需要修改的评价
  2. 修改评价内容
  3. 保存修改

### 2.3 查看自我评价

- **操作角色**：学生、企业管理员、企业导师、班主任、学校管理员
- **操作说明**：
  1. 进入评价列表页面
  2. 查看评价详情

## 五、综合评分

### 5.1 自动计算

- **触发条件**：当自我评价、企业评价、学校评价都完成后
- **计算方式**：系统根据配置的权重自动计算
- **计算公式**：
  ```
  综合评分 = 自我评价 × 权重1 + 企业评价 × 权重2 + 学校评价 × 权重3
  ```
- **权重配置**：由系统管理员在系统配置中设置

### 5.2 查看综合评分

- **操作角色**：学生、企业管理员、企业导师、班主任、学校管理员
- **操作说明**：
  1. 进入综合评分页面
  2. 查看评分详情：
     - 各项评价得分
     - 综合评分
     - 评分明细

## 六、评价状态说明

| 状态码 | 状态名称 | 说明 |
|--------|---------|------|
| 0 | 待提交 | 评价尚未提交 |
| 1 | 已提交 | 评价已提交，等待确认 |
| 2 | 已确认 | 评价已确认，不能修改 |
| 3 | 已计算 | 综合评分已计算 |

## 七、权限说明

### 7.1 学生
- 提交自我评价
- 修改自己的自我评价（在允许的时间范围内）
- 查看自己的所有评价和综合评分

### 7.2 企业管理员
- 提交本企业所有学生的企业评价
- 修改本企业的评价（在允许的时间范围内）
- 查看本企业所有学生的评价和综合评分

### 7.3 企业导师
- 提交分配给自己的学生的企业评价
- 修改分配给自己的学生的评价（在允许的时间范围内）
- 查看分配给自己的学生的评价和综合评分

### 7.4 班主任
- 提交本班级学生的学校评价
- 修改本班级学生的评价（在允许的时间范围内）
- 查看本班级所有学生的评价和综合评分

### 7.5 学校管理员
- 提交所有学生的学校评价
- 修改所有学生的评价（在允许的时间范围内）
- 查看所有学生的评价和综合评分
- 配置评价权重

## 九、常见问题

### Q1: 评价可以修改吗？
A: 在允许的时间范围内可以修改，但确认后不能修改。

### Q2: 综合评分什么时候计算？
A: 当自我评价、企业评价、学校评价都完成后，系统自动计算。

### Q3: 评价权重如何设置？
A: 由系统管理员在系统配置中设置，一般默认权重为：自我评价30%，企业评价50%，学校评价20%。

### Q4: 可以只提交部分评价吗？
A: 可以，但只有所有评价都完成后，才会计算综合评分。

### Q5: 评价有时间限制吗？
A: 一般要求在实习结束后一定时间内提交，具体时间由系统配置决定。

## 十、注意事项

1. **评价及时性**：评价应在实习结束后及时提交，避免影响综合评分计算。
2. **评价客观性**：评价应客观、公正，真实反映学生的实习表现。
3. **评价完整性**：各项评价指标都应填写完整，避免影响综合评分。
4. **权重配置**：评价权重应合理配置，确保综合评分的准确性。
5. **数据权限**：不同角色只能操作权限范围内的评价数据。
